# 🐾 PetShare Israel — Telegram-маркетплейс услуг с животными

Многоязычный Telegram-бот-маркетплейс, объединяющий владельцев животных и
клиентов в Израиле: аренда животных для событий, конные прогулки,
передержка, знакомства питомцев и доска объявлений — с модерацией,
автоматическим расчётом комиссий и учётом всех сделок в Google Sheets.

Работает в продакшене 24/7 на VPS (Ubuntu, systemd).

---

## Что умеет бот

| Услуга | Для клиента | Для владельца | Монетизация |
|---|---|---|---|
| 🐾 **Аренда животных** | каталог с фото и рейтингами, фильтры, заявка за 2 минуты | анкета питомца с модерацией | 10% сбор + 20% комиссия |
| 🐴 **Конные прогулки** | выбор времени суток (закатный тариф — дороже), кол-во человек, уровень опыта | регистрация лошади с двумя тарифами | 10% + 20% |
| 🏡 **Передержка** | заявка с датами и особенностями ухода | анкета пет-ситтера, каталог | 10% + 20% (суток × тариф) |
| 💞 **Знакомства питомцев** | «тест-драйв породы», прогулки, вязка | анкета с документами | сбор за вязку, дружба бесплатно |
| 📢 **Доска объявлений** | продам / ищу, связь через бота | счётчик приведённых клиентов | готовый переключатель платного размещения |

Плюс: 🐶 **квиз «Подбери породу»** (5 вопросов → топ-3 породы → лид в
таблицу спроса), 📋 личный кабинет заявок, ⭐ отзывы после подтверждённых
сделок с рейтингом в карточках, ✉️ переписка покупатель-продавец через
бота (контакты не раскрываются, администратор видит копии).

## Ключевые особенности

- **3 языка интерфейса** — русский, английский, иврит; выбор при первом
  запуске, хранится постоянно; уведомления каждому участнику сделки
  приходят на его языке
- **Автоперевод контента** — свободные тексты карточек переводятся
  ru→en/he автоматически (Google Translate), словарные значения (города,
  статусы, категории) — через локальный словарь
- **Модерация всего** — каждая анкета и объявление проходят через
  администратора: кнопки ✅/❌ прямо в уведомлении, статусы в таблице
- **Деньги считаются сами** — по каждой заявке бот пишет в таблицу цену,
  сбор с клиента, комиссию владельца и доход платформы
- **Маркетинг измерим** — deep-links с метками источников
  (`?start=insta`), лист «Маркетинг», карта спроса из квиза и «пустых»
  экранов
- **Устойчивость** — кэширование Google Sheets, ловец «потерянных»
  сообщений (бот никогда не молчит), таймауты анкет, systemd-автозапуск
  с рестартом при сбое

## Архитектура

```
Telegram ⟷ bot.py (python-telegram-bot 22, polling)
              │  12 диалогов-анкет (ConversationHandler)
              │  i18n.py — 200+ строк перевода × 3 языка
              ▼
          sheets.py — слой данных с кэшем (TTL 5 мин)
              ▼
        Google Sheets — 12 листов: животные, заявки, владельцы,
        ситтеры, передержка, лошади, конные заявки, знакомства,
        объявления, отзывы, спрос, маркетинг
```

- **Хранилище** — Google Sheets: заказчик управляет данными в привычной
  таблице, без админ-панели и SQL
- **Состояние пользователей** (язык, счётчики) — PicklePersistence,
  переживает перезапуски
- **Тесты** — 51 pytest-тест чистой логики: комиссии, фильтры, подбор
  пород, рейтинги, переводы (кэш подменяется, сеть не нужна)

## Стек

Python 3.12+ · python-telegram-bot 22 (async) · gspread / Google Sheets
API · deep-translator · pytest · systemd / launchd · Ubuntu VPS

## Структура проекта

```
bot.py          — обработчики: меню, каталоги, 12 анкет, админ-решения
i18n.py         — переводы интерфейса (ru/en/he) и словарь значений
sheets.py       — работа с Google Sheets, кэш, генерация ID
config.py       — настройки из .env
setup_sheets.py — разовая инициализация структуры таблицы
tests/          — 51 тест бизнес-логики
deploy/         — systemd-юнит, скрипт деплоя одной командой, инструкция
```

## Запуск

```bash
python3 -m venv venv && ./venv/bin/pip install -r requirements.txt
cp .env.example .env        # вписать токен бота, ID админа и таблицы
# + положить service_account.json (сервисный аккаунт Google)
./venv/bin/python setup_sheets.py   # создать листы с демо-данными
./venv/bin/python bot.py
```

Деплой на сервер — одной командой: `bash deploy/to_vps.sh <IP>`
(подробности в [deploy/DEPLOY.md](deploy/DEPLOY.md)).
